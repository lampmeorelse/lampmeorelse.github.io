HermitBot Docs

Command Usage Description
>HELP >HELP Sends a link to this page.
>RULES >RULES Shows the current Discord Server's rules.
>IP >IP Says the IP for the BlockHermit Minecraft Server.
>SENIORS >SENIORS Says a short blurb about the Seniors of BlockHermit, and what they are.
>LMGTFY >LMGTFY [QUERY] Replies with a link to the LMGTFY page for the query entered.
>GOOGLE >GOOGLE [QUERY] Replies with a link to the Google page for the query entered.
>YOUTUBE >YOUTUBE [QUERY] Replies with a link to the YouTube page for the query entered.
>SOUNDS >SOUNDS [SOUND] Plays the stated sound in a voice channel.
>CRAFT >CRAFT [MC ITEM] Replies with the crafting recipe for the entered item.
>CARD >CARD Replies with a randomly selected playing card!
>CHOOSE >CHOOSE [OPTIONS SEPARATED BY COMMAS] Chooses an option for you!
>8BALL >8BALL [QUESTION] Answers a question 8-Ball style!
>CAT >CAT Returns a funny cat picture or gif!
