Command Usage Description
>RULES >RULES Shows the rules of the BlockHermit Discord server.
>IP >IP Shows the Official BlockHermit Server's IP Address.
>SENIORS >SENIORS Shows information about the Seniors of BlockHermit.
>LMGTFY >LMGTFY [SEARCH QUERY] Replies with a link to a LMGTFY search.
>GOOGLE >GOOGLE [SEARCH QUERY] Replies with a link to a Google search.
>YOUTUBE >YOUTUBE [SEARCH QUERY] Replies with a link to a YouTube search.
>SOUNDS >SOUNDS [SOUND] Plays the sound said by the user.
>CRAFT >CRAFT [MC ITEM] Replies with a picture of how to craft the item input.
>CARD >CARD Returns a picture and link to a randomly drawn playing card.
>CHOOSE >CHOOSE [OPTIONS SEPARATED BY COMMAS] Chooses an option for you.
>8BALL >8BALL [QUESTION] Answers a question 8-Ball style!
>PROFILE >PROFILE [MENTION OF A USER] Returns the currently available profile information on a user.
>CAT >CAT Returns a random image of a cat.
